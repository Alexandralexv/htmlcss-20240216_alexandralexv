Modal title

Модальные окна сделаны на HTML, CSS и JavaScript. Они располагаются поверх всего документа и блокируют прокрутку тела документа, вместо него прокручивая модальные окна. Клик вне модального элемента автоматически закрывает его. Bootstrap позволяет активировать лишь один модальный элемент за раз.
